## 使用示例

### 启动服务器
```bash
python webrtc_server_for_http.py
```

### 访问页面
- 视频通话：http://localhost:8765/webrtc_client.html
- 状态监控：http://localhost:8765/status.html
- 服务器状态 API: http://localhost:8765/status
- 房间信息 API: http://localhost:8765/api/rooms

### 典型使用流程
1. 打开浏览器访问 webrtc_client.html
2. 点击"连接信令服务器"
3. 在右侧查看在线用户列表
4. 点击某个用户的"呼叫"按钮
5. 对方收到呼叫请求，点击"接听通话"
6. 建立视频通话
7. 可以开关音频/视频
8. 通话结束点击"挂断通话"

## 技术细节

### WebRTC 信令流程
```
主叫方                    信令服务器                    被叫方
  |                          |                          |
  |---call-request--------->|---call-request--------->|
  |                          |                          |
  |                          |<--call-accepted--------|
  |<--call-accepted--------|                          |
  |                          |                          |
  |------offer------------->|------offer------------->|
  |                          |                          |
  |<-----answer------------|<-----answer------------|
  |                          |                          |
  |---ice-candidate------->|---ice-candidate------->|
  |                          |                          |
  |<--ice-candidate-------|<--ice-candidate-------|
  |                          |                          |
  |==== WebRTC P2P 连接建立 ====|
```

### 消息类型对照表

| 消息类型 | 方向 | 说明 |
|---------|------|------|
| register | C->S | 客户端注册 |
| welcome | S->C | 欢迎消息 |
| get-peer-list | C->S | 获取用户列表 |
| peer-list | S->C | 用户列表 |
| peer-joined | S->C | 用户加入通知 |
| peer-left | S->C | 用户离开通知 |
| call-request | C->S / S->C | 呼叫请求 |
| call-accepted | C->S / S->C | 呼叫接受 |
| call-rejected | C->S / S->C | 呼叫拒绝 |
| offer | C->S / S->C | SDP Offer |
| answer | C->S / S->C | SDP Answer |
| ice-candidate | C->S / S->C | ICE 候选 |
| hangup | C->S / S->C | 挂断通话 |
